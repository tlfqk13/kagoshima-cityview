### QA에서 발견·수정된 이슈

- **ISS-004 (Critical):** NextAuth `UntrustedHost` 시 인증 fail-open → `/admin` 무인증 노출. `trustHost: true`로 수정.
- **ISS-005 (High):** Node의 `navigator.language='en-US'`를 언어 감지가 읽어 SSR/클이언트 언어 불일치(React #418). SSR 언어 ko 고정 + 마운트 후 전환으로 수정.
