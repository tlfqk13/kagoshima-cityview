## ケース別の対応

### 1. QRを読んでもページが開かない・表示されない

1. まずご利用者様の通信環境(機内モード・Wi-Fi)をご確認ください
2. それでも開かない場合は、運営側の障害の可能性があります。**fkffksk20@gmail.com までご連絡ください — 24時間以内に回答・復旧対応します**
3. 復旧までの間、停留所の時刻表は鹿児島市交通局の公式案内(https://www.kotsu-city-kagoshima.jp)をご案内ください
4. なお運営側では監視サービスによる24時間監視を行っており、障害は運営側でも即時に検知します

### 2. 表示される位置が違う気がする

- お手数ですが fkffksk20@gmail.com まで停留所番号(No.)と内容をお知らせください。公式データ(鹿児島市交通局公表値)と照合の上、24時間以内に回答します

### 3. QRが読み取れない (カードの汚損・退色)

- カード下部のURLを直接入力しても同じページが開きます
- 交換用カードは https://www.kagoshima-cityview.com/card からいつでも再印刷できます(無料)

### 4. カードの設置をやめたい

- いつでも・ご連絡なしでお下げいただけます。報告も不要です
