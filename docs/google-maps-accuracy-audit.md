# 구글맵 정류장 위치 정확도 감사 (2026-07-20)

> 가고시마 시티뷰 버스 전 20개 정류장에 대해, 구글맵이 안내하는 위치와 공식 GTFS+현장 검증 좌표를 비교한 감사 기록. 관광과 제안의 핵심 증거 자료.

## 1. 목적

"기존 지도 서비스가 정류장 위치를 잘못 표시한다"는 서비스의 존재 근거를 일화(anecdote)가 아닌 **전수 조사 데이터**로 입증한다.

## 2. 방법론

- **쿼리:** 각 정류장의 기본 일본어명(방면 괄호 제거) + `バス停 鹿児島` — 정류장 앞에서 버스를 찾는 관광객의 검색 행동 모사 (`hl=ja`)
- **수집:** 헤드리스 Chrome으로 검색 결과 DOM 덤프. 구글이 장소 페이지로 직행하는 경우 CDP로 최종 URL의 `@위도,경도` 취득
- **매칭:** 검색 후보 중 정류장명과 이름 유사도가 가장 높은 후보를 선택 (유사도 기재)
- **오차:** haversine 거리(m). 공식 좌표 = `src/data/routes/cityview.json` (GTFS 가공 + 현장 GPS 검증)
- **판정 기준:** <50m 정상 · 50~100m 주의 · ≥100m 오류
- **도구:** `scripts/google-maps-audit/` (collect.sh → collect-place-pages.sh → parse.py)

### 주의사항 (해석 전제)

- 단일 시점(2026-07-20), 단일 실행, 한국 IP에서 수집 — 구글 결과는 시기·지역·세션에 따라 변동될 수 있음
- "이름 유사도 최고 후보"는 관광객이 실제로 탭하는 결과와 다를 수 있음
- 50m 미만은 스마트폰 GPS 수신 오차 범위와 겹치므로 사실상 동일 위치로 간주 가능

### 기준 좌표 검증: 교통국 공식 API와 전건 일치 (2026-07-20)

감사의 "정답 기준"인 `cityview.json` 좌표를 가고시마시 교통국의 공식 시각검색 시스템과 교차 검증했다.

- **공식 출처:** 교통국 `strEnd.php?rosen_id=1680` API (`docs/data-update-guide.md`에 문서화된 공식 추출 경로. 야경 코스 좌표 출처와 동일 시스템)
- **방법:** 공식 20개 엔트리와 우리 20개 정류장을 이름 정규화(전각/반각 괄호 혼용, 巖/巌 한자 변형 흡수) 후 매칭, haversine 비교
- **결과:** **20/20 전건 Δ0.000m** — 방면 변형(城山向け / 仙巌園方面向け) 포함, 소수점 8자리까지 완전 일치
- **의미:** 본 감사의 기준 좌표는 교통국이 현재 공표하는 값과 전건 일치한다. 따라서 아래 표의 오차는 기준의 불확실성이 아니라 지도 서비스 측의 표시 오차다.
- **비고:** 공식 측 데이터에도 명칭 변형이 존재(半角/全角 괄호 혼용, `仙巖園`의 巖 구字形) — GTFS 갱신 대조 시 유의.

## 5. 케이스별 주석

- **stop_01/20 (164m):** 가고시마 중앙역은 승강장이 많은 대형 정류장. 구글의 "鹿児島中央駅 バス停"은 시티뷰 승강장에서 164m 떨어진 곳을 가리켜, 관광객이 반대편 출구로 유도됨. EP.01의 사례와 같은 "정류장은 있는데 못 찾는" 실패 유형.
- **stop_03 (86m):** 데브로그 EP.01의 원래 사례. 2026-05 현장 확인 당시 약 200m였으나 2026-07 현재 86m로 부분 수정됨. **구글 데이터는 무단으로 수시 변경되므로 검증 없이는 신뢰할 수 없다는 사실을 오히려 입증.**
- **stop_13 (220m):** 본 감사 최대 오차. 이진칸·이소 해수욕장 방면 관광객에게 치명적.
- **stop_14 (55m):** 구글에 `(シティビュー)`라는 정확한 노선 전용 엔트리가 있음에도 좌표가 어긋난 사례 — 이름만 맞으면 되는 게 아님을 보여줌.
- **stop_16 (7m):** 위치는 정확하나 명칭이 구글 "かごしま水族館(下り)" vs 공식 "水族館前（桜島桟橋）"로 달라, 페리 이용객이 방면을 혼동할 수 있는 유형.
- **정상 13곳 중에도 10~39m:** 스마트폰 GPS 오차 범위와 겹침. 구글도 "대체로 맞지만 항상 검증된 것은 아님"이 전수 조사의 결론.

## 6. 제안서용 일본어 요약

> 2026年7月の調査において、鹿児島シティビューバス全20停留所についてGoogleマップの表示位置を検証したところ、**7停留所(35%)で50m以上、うち3停留所(鹿児島中央駅・異人館前・鹿児島駅前)で100m以上の位置ずれ**(最大220m)を確認しました。また過去には約200mの誤表示が後日修正された事例もあり、地図サービスの表示は予告なく変動するため、常に検証された案内が必要であることが分かりました。なお比較の基準とした停留所座標は、鹿児島市交通局の公表情報と全20停留所で一致していることを同日に確認しています。

## 7. 재현 방법

```bash
scripts/google-maps-audit/collect.sh                       # 1단계: DOM 수집
scripts/google-maps-audit/collect-place-pages.sh stop_XX   # 1-B: 장소 페이지형 정류장
python3 scripts/google-maps-audit/parse.py                 # 2단계: 오차 계산·표 출력
```

결과 JSON은 `scripts/google-maps-audit/out/audit.json`에 저장. 정기 점검(분기) 시 동일 절차로 재실행하고 본 문서의 표와 일자를 갱신한다.
